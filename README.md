# 3155_Final

The 3155 final project

## Run Locally

Clone the project

```bash
  git clone https://github.com/JuanPablo2655/3155_Final.git
```

Go to the project directory

```bash
  cd 3155_Final
```

Create a virtual environment

```bash
  python -m venv venv
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  flask --debug run
```
